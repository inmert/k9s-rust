# K9S-Rust Project
A terminal-based Kubernetes cluster manager inspired by k9s, built with Rust.

## Contributing

This is a learning project, but contributions are welcome! Areas that could use help:

1. Implementing additional resource views
2. Adding tests
3. Performance optimization
4. UI/UX improvements
5. Documentation
